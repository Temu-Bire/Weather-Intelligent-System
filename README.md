# Weather Intelligent

A smart weather application with intelligent analysis and recommendations. Built with Python (aiohttp) backend and vanilla JavaScript frontend.

## Features

- Real-time weather data fetching
- Intelligent weather analysis and recommendations
- Historical weather data storage
- Clean and responsive frontend
- RESTful API


## Tech Stack

- **Backend**: Python 3, aiohttp, asyncpg, Pydantic
- **Database**: PostgreSQL
- **External API**: OpenWeatherMap
- **Frontend**: HTML, CSS, JavaScript (Vanilla)
